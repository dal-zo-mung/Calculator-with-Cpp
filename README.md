# Calculator-with-Cpp

C++ Logic-Testing Calculator
A simple, interactive command-line calculator built to practice Data Structures (Structs) and Flow Control in C++.

# Features

Custom Struct: Uses a calculator struct to bundle operand data and operation symbols.
Continuous Loop: Runs until the user explicitly types out.
Basic Math: Supports addition, subtraction, multiplication, and division.
Power Function: Includes exponentiation using the <cmath> library.
Error Handling: (In Progress) Basic checks for division by zero.

# Tech Stack

Language: C++
Concepts: Structs, While-loops, If-Else logic, Standard I/O.

# How to Use

Compile the code using any C++ compiler (e.g., g++ main.cpp -o calculator).
Run the executable.
Type start to begin a calculation or out to exit.
Enter your numbers and follow the prompts for the operator.
